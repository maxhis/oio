
## Build

```bash
pnpm build
pnpm preview
```

## Cloudflare Pages

Use these build settings in Cloudflare Pages:

- Build command: `pnpm build`
- Build output directory: `dist`

SPA routing fallback is handled by [`public/_redirects`](/Users/istar/proj/oio/public/_redirects).
